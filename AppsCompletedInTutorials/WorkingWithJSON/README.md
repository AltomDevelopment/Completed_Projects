# Working with JSON
---

1. Introduction
   - JSON stands for Javascript Object Notation
   - It is deverified from Java Script but not depending on Javascript
   - JSON data is lightweight and human readable text
   - JSON file names are extension with .json
   - JSON Internet Media  Type is "application/json"

2. What are you trying to solve ("Actually sending JSON to update the Web Page")

Example
{
    "name": "Dipesh",
    "age": 30,
    "isProgrammer": true,
    "skills": ["html", "css", "javascript"],
    "address": {
        "city": "Mumbai",
        "state": "MH"
    }
}
